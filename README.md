# Project Dashboard

> A personal dashboard I built to track my own projects — aggregates and showcases projects with live GitHub stats.

## Features

- 📊 Live project cards with GitHub stats (stars, forks, last updated)
- 🔍 Filter by tech stack
- 🌙 Dark theme
- 📱 Responsive grid layout

## Setup

```bash
npm install
npm run dev
```

Edit `src/projects.js` to configure your own projects.

## Why I built this

I wanted a single dashboard to track all my portfolio projects in one place. This pulls live stats from the GitHub API so I can see stars, forks, and activity at a glance.
